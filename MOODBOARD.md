## Typography

### Recommended Font Pairings

**Option A: Modern & Bold**
```
Headings:  Inter (700/800 weight)
           - Clean, modern, highly readable
           - Strong presence without being flashy

Body:      Inter (400/500 weight)
           - Consistent family throughout
           - Excellent screen readability

Accent:    Playfair Display (for quotes/special text)
           - Elegant serif for contrast
```

**Option B: Byzantine Influence**
```
Headings:  Cinzel (600/700 weight)
           - Roman/Byzantine inspired capitals
           - Dignified, traditional feel

Body:      Source Sans Pro (400)
           - Clean, neutral, highly readable
           - Good pairing with decorative headers

Accent:    Cormorant Garamond (italic)
           - Classical elegance for quotes
```

**Option C: Contemporary Orthodox**
```
Headings:  Oswald (500/600 weight)
           - Condensed, bold, modern
           - Strong visual impact

Body:      Lato (400)
           - Warm, friendly, readable
           - Humanist sans-serif

Accent:    EB Garamond (for liturgical text)
           - Traditional feel for quotes
```

### Typography Scale

```
Display (Hero):     4rem / 64px    - Church name, major headings
H1:                 2.5rem / 40px  - Page titles
H2:                 1.875rem / 30px - Section headings
H3:                 1.5rem / 24px  - Subsection headings
H4:                 1.25rem / 20px - Card titles
Body:               1rem / 16px    - Main content
Small:              0.875rem / 14px - Captions, metadata
```

## Photography Style

### Preferred Image Types

1. **Church Interior**
   - Iconostasis
   - Candlelight/natural lighting
   - Liturgical scenes

2. **Community**
   - Congregation at worship
   - Fellowship events
   - Outreach activities

3. **Orthodox Imagery**
   - Icons (high quality reproductions)
   - Liturgical objects
   - Architecture details

### Image Treatment

```
┌─────────────────────────────────────────────────┐
│                                                 │
│  ○ Warm, natural tones                          │
│  ○ Slight desaturation for cohesion             │
│  ○ Good contrast, not overly processed          │
│  ○ Respectful, dignified compositions           │
│                                                 │
│  AVOID:                                         │
│  ✗ Stock photo "church" clichés                 │
│  ✗ Overly bright/saturated                      │

│  ✗ Casual/irreverent imagery                    │
│                                                 │

```

## Inspiration References

### Websites with Similar Aesthetic

1. **Ancient Faith Ministries** - ancientfaith.com
   - Clean, modern Orthodox design
   - Good use of imagery and typography

2. **Greek Orthodox Archdiocese** - goarch.org
   - Traditional yet accessible
   - Clear navigation

3. **Hallow App** - hallow.com
   - Modern religious app design
   - Bold typography, dark theme option

4. **EWTN** - ewtn.com
   - Media-focused religious site
   - Live streaming integration

### Design Trends to Embrace

- High contrast dark/light sections
- Bold, confident typography
- Generous whitespace
- Card-based layouts
- Subtle animations (scroll reveals, hover states)
- Mobile-first responsive design

### Trends to Avoid

- Overly "churchy" clip art
- Dated web design patterns
- Busy, cluttered layouts
- Generic stock photography
- Trendy effects that won't age well
